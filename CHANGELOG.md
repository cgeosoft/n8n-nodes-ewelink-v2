## [0.1.0] - 2026-02-05

### Added
- Initial release of n8n-nodes-ewelink-v2
- Support for eWeLink v2 API with OAuth2 authentication
- Device operations: Get Many, Get, Get Status, Set Status, Get History, Update, Delete, Share
- Group operations: Get Many, Create, Set Status, Update, Delete
- Home operations: Get Families, Create Family, Update Family, Delete Family, Add Room, Delete Room, Get Home Page
- Multi-channel device support
- CLI OAuth helper tool for easy token generation
- Comprehensive documentation and examples
